üß™ Testing Instructions

How to Test the Extension

  1. Load the Extension:
    • Open Chrome and navigate to chrome://extensions
    • Enable "Developer mode" (top right)
    • Click "Load unpacked"
    • Select the dist/ folder
  2. Verify Extension Loaded:
    • Look for "ShieldPro Ultimate" in extensions list
    • Check that the extension icon appears in toolbar
    • Click the icon to open the popup
  3. Test Torrent Site Blocking:
    • Visit any torrent site (e.g., 1337x.to)
    • Observe that ads are blocked
    • Check that no popups appear
    • Verify clicking doesn't trigger ad redirects
  4. Check Console for Blocking Messages:
    • Open Developer Tools (F12)
    • Go to Console tab
    • Look for "[ShieldPro]" messages
